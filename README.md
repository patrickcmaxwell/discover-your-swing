# Discover Your Swing

Next.js rebuild for Discover Your Swing, an Altitude client site.

## Source capture

The original WordPress/Elementor content was captured from:

- `https://discoveryourswing.com/wp-json/wp/v2/pages`
- `https://discoveryourswing.com/wp-json/wp/v2/posts`
- `https://discoveryourswing.com/wp-json/wp/v2/media`
- `https://discoveryourswing.com/`

Raw captures are stored in `scrape/`. Brand and page imagery used by the draft is stored in `public/images/`.

## Local development

```bash
npm install
npm run dev
```
